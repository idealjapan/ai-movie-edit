## 🎯 起動確認チェックリスト

### ✅ Windows
- [ ] Docker Desktopのクジラアイコンが緑色
- [ ] コマンドプロンプトにエラーが出ていない
- [ ] ブラウザで localhost:8501 が開ける

### ✅ Mac
- [ ] メニューバーにクジラアイコンが表示
- [ ] ターミナルにエラーが出ていない
- [ ] ブラウザで localhost:8501 が開ける

---

## 🚨 エラーが出た時の対処法

### 「Docker Desktop is not running」と表示される
→ Docker Desktopを起動してください

### 「ファイルが見つかりません」と表示される
→ cdコマンドで正しいフォルダに移動してください

### 「Permission denied」と表示される
→ 管理者権限で実行してください

## 💡 便利な使い方

### 毎日の起動を簡単に（Windows）
デスクトップの「AI動画編集ツール.bat」のショートカットを作成

### 毎日の起動を簡単に（Mac）
ターミナルのコマンドをメモアプリに保存しておく：
```
cd Desktop/ai-movie-edit-main
docker-compose up
```
